# 台灣線上教學組合包

這是一個專為台灣學校線上教學所設計的專案，目標是透過 <a href="https://discord.com/brand-new">Discord</a> 這款成熟的線上語音軟體，讓老師可以輕鬆建立線上教學環境，達成線上點名、學生管理、作業繳交等功能。  

- <a href="/">Youtube 教學系列</a> (製作中)
- <a href="/">線上維基百科</a> (製作中)
